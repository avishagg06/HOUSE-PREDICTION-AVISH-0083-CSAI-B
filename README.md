## 📌 Project Overview

The House Price Prediction project is a Machine Learning application that predicts the price of a house based on various features such as area, number of bedrooms, bathrooms, location, and other property-related attributes. The model helps buyers, sellers, and real estate businesses estimate property prices accurately.

---

## 🎯 Objective

- Predict house prices using Machine Learning.
- Perform data cleaning and preprocessing.
- Train and evaluate regression models.
- Assist users in estimating property values.

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset

The dataset contains property-related information such as:

- Area (Square Feet)
- Number of Bedrooms
- Number of Bathrooms
- Floors
- Location
- Parking Availability
- Furnishing Status
- House Price (Target Variable)

---

## ⚙️ Project Workflow

1. Import required libraries.
2. Load the dataset.
3. Perform data cleaning.
4. Handle missing values.
5. Encode categorical variables.
6. Split the dataset into training and testing sets.
7. Train the regression model.
8. Evaluate model performance.
9. Predict house prices for new data.
